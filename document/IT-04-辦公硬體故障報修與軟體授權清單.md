# IT 部門：硬體報修與軟體安裝指南

## 1. 電腦報修流程
若發生硬體故障（如螢幕損壞、電池膨脹、無法開機）：
1. 登入企業 Portal 填寫報修單。
2. 將設備攜至 IT 櫃台進行檢修。
3. 若需備援機，請由主管核准後領用。

## 2. 標準軟體清單
員工可自行透過「軟體中心 (Software Center)」安裝以下授權軟體：
- Microsoft 365 辦公套件
- Slack / Microsoft Teams
- JetBrains 全系列 (需申請 License)
- Docker Desktop (限研發單位)

## 3. 嚴禁事項
嚴禁私自安裝未經授權之盜版軟體或 P2P 下載軟體，違者將由資安小組通報懲處。
