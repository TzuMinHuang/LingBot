# IT 部門：開發資源申請流程

## 1. 虛擬機 (VM) 申請
開發人員如需申請獨立開發環境，請至 ODS 系統填寫「伺服器資源申請單」。
- **基本配備**：4 vCPU, 8GB RAM, 100GB SSD。
- **作業系統**：預設為 Ubuntu 22.04 LTS。

## 2. 容器環境 (K3s/Docker)
公司提供統一的測試環境 Namespace。
- **Registry 地址**：harbor.internal.company.com
- **部署規範**：所有部署映像檔須通過 Trivy 漏洞掃描。
- **資源限制 (Quotas)**：單一 Namespace 預設限制為 CPU 2C / Memory 4Gi。

## 3. 資料庫申請
目前支援 PostgreSQL 15 與 Redis 7 的自動化部署申請，請註明所需容量與存取 IP 白名單。
